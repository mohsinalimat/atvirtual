El apoyo MIoT a la Formación

Salvamento Marítimo lanza FormaciON, un ambicioso proyecto de innovación con el que se busca incorporar el Internet de las Cosas para mejorar la comunicación entre alumnos e Instructores durante las prácticas.

Objetivo

El fin del proyecto es la gestión con Internet de las Cosas (Managemente with IoT - MIoT) de las comunicaciones o elementos de las Sesiones de Formación donde los participantes llevan dispositivos o las áreas disponen de equipamiento para mejorar el proceso de aprendizaje, transmitir información de forma clara y recolectar datos importantes.

Plataforma

Basada en software y librerías de código abierto, completamente autónoma y segura, sobre una aplicación web responsiva y adaptable a cualquier dispositivo.

Optimizada para los navegadores Chrome, Safari, Edge y Firefox.
